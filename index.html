Zero G Arena

Zero G Arena is a fully released PC game that I developed over the course of about three and a half years. It's an online multiplayer competetive shooter game with unique zero gravity mechanics.

Zero G Arena on Steam

Technology:

  • Unreal Engine 4
  • C++
  • Blueprint Visual Scripting

Achievements:

  • Building an AI from scratch to accomodate ZGA's unique game mechanics. This involved creating custom navigation systems.
  • Implementing networked gameplay.
  • Finishing a fully-featured comercial game project. Happily, the game was well received by players with a review score of 90% at the time of writing (which peaked at 97% when the game had a more active playerbase).
  • Acting on player feedback to improve the game over the course of 2 years in Early Access.

Space Game

My first non-trivial programming project. It's a (pretty generic) 2d top down space shooter, not unlike the classic arcade game "Asteroids". This was only ever supposed to be a learning project, but it was extremely effective for learning C++ and game programming in general.

The comparitively low-level nature of SFML (as compared to game engines like UE4) provided a great introduction to a lot of the more low-level problems in game development.
